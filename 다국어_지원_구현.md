# 다국어 지원 (한글/영어) 구현 완료

## 구현 내용

포트폴리오 웹사이트에 한글과 영어 간 전체 페이지 언어 전환 기능을 성공적으로 구현했습니다.

## 주요 변경사항

### 1. 라이브러리 설치
- `i18next` 및 `react-i18next` 패키지 설치
- React 애플리케이션에 최적화된 국제화(i18n) 라이브러리

### 2. 번역 파일 생성
**위치**: `/src/i18n/locales/`

- **ko.json**: 한국어 번역
- **en.json**: 영어 번역

모든 페이지의 텍스트 콘텐츠를 포함:
- 네비게이션 메뉴
- 프로필 정보
- 경력 사항
- 프로젝트 설명
- 학력 및 자격증
- 푸터

### 3. i18n 설정 파일
**파일**: `/src/i18n/config.ts`

- 기본 언어: 한국어 (ko)
- 지원 언어: 한국어, 영어
- 자동 초기화 설정

### 4. 언어 전환 버튼 컴포넌트
**파일**: `/src/app/components/LanguageToggle.tsx`

- Languages 아이콘과 함께 표시
- 현재 언어에 따라 "EN" 또는 "한글" 표시
- 클릭 시 즉시 언어 전환

### 5. 페이지 업데이트

#### HomePage (`/src/app/pages/HomePage.tsx`)
- 네비게이션 메뉴 번역 적용
- 언어 전환 버튼 추가
- 푸터 텍스트 번역 적용

#### CareerPage (`/src/app/pages/CareerPage.tsx`)
- 모든 섹션 제목 번역
- 경력 사항 및 프로젝트 설명 번역
- 학력, 자격증, 어학 점수 번역
- 언어 전환 버튼 추가

### 6. 메인 앱 초기화
**파일**: `/src/main.tsx`

- i18n 설정 파일 import 추가
- 앱 시작 시 자동으로 i18n 초기화

## 사용 방법

1. **홈페이지에서**:
   - 상단 네비게이션 바의 언어 전환 버튼 클릭
   - "EN" 버튼을 클릭하면 영어로 전환
   - "한글" 버튼을 클릭하면 한국어로 전환

2. **경력기술서 페이지에서**:
   - 상단 네비게이션 바의 언어 전환 버튼 클릭
   - 전체 이력서 내용이 선택한 언어로 즉시 변환

## 기술적 특징

- ✅ 전체 페이지 실시간 언어 전환
- ✅ 라우트 간 언어 설정 유지
- ✅ TypeScript 타입 안정성 확보
- ✅ 깔끔한 UI/UX (아이콘 + 텍스트)
- ✅ 모든 컴포넌트에서 일관된 번역 적용

## 테스트 결과

브라우저 테스트를 통해 다음 사항을 확인했습니다:

1. ✅ 홈페이지에서 언어 전환 정상 작동
2. ✅ 경력기술서 페이지에서 언어 전환 정상 작동
3. ✅ 페이지 간 이동 시 언어 설정 유지
4. ✅ 모든 텍스트 콘텐츠 정상 번역
5. ✅ 버튼 레이블 동적 업데이트

## 향후 확장 가능성

필요 시 다음과 같은 기능을 쉽게 추가할 수 있습니다:

- 추가 언어 지원 (일본어, 중국어 등)
- 브라우저 언어 설정 자동 감지
- 로컬 스토리지에 언어 설정 저장
- URL 파라미터를 통한 언어 설정
